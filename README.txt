Kanban
======

Mettre en place la méthode Kanban avec des threads et une synchronisation sur réveil selon différentes conditions

Set up the Kaban's method by using threads and synchronization on awakening according to the conditions

======

  Comment démarrer le programme :
Depuis la console, se placer dans le bon dossier.
Ensuite, entrer la commande "make all clean" pour créer l'exécutable appelé "main".
Une fois créé, démarrer le programme avec "./main <nb Postes de travail>" ;
le nombre de postes de travail doit être supérieur ou égal à 1. 
Enfin, comme demandé par le programme, donner le nombre de pièces à fabriquer.


  How to start the program :
From the terminal, take place in the good folder. 
Then, write the "make all clean" command to create the feasible named "main".
Once created, start the program with "./main <nb Workshop>" ; 
the number of workshops have to be greater or equal than 1.
Finally, as requested by the program, give the number of pieces to be made.
